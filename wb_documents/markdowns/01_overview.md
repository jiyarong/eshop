# WildBerries API 概述

## 简介

WB API 是基于 HTTP REST 协议的卖家管理 API，提供：商品管理、订单处理、营销推广、数据分析、用户沟通、财务报表等功能。文档以 Swagger OpenAPI 格式提供，可导入 Postman 或用 CodeGen 生成客户端代码。

## 认证

### Token 类型

| 类型 | 适用场景 | 有效期 |
|------|----------|--------|
| **Personal（个人）** | 卖家自用 | 180 天 |
| **Service（服务）** | 第三方服务商 | 180 天 |
| **Basic（基础）** | 低频场景 | 180 天 |
| **Test（测试）** | Sandbox 测试 | 180 天 |

### 使用方式

在请求 Header 中加入：
```
Authorization: <your_token>
```

Token 按 API 类别划分权限，每个 Token 只能访问其对应类别的接口（如 Content Token 只能访问商品接口）。

## HTTP 状态码

| 状态码 | 含义 |
|--------|------|
| 200 | 成功 |
| 204 | 删除/更新/添加成功 |
| 400 | 请求参数错误 |
| 401 | 未授权（Token 无效/过期/缺失） |
| 402 | 余额不足（仅限业务解决方案目录的服务） |
| 403 | 访问被禁（Token 权限不足，或功能需要 Джем 订阅） |
| 404 | 路径不存在 |
| 409 | 部分数据保存失败（计为 5~10 次请求） |
| 413 | 请求体超过数据量限制 |
| 422 | 参数逻辑冲突 |
| 429 | 请求频率超限 |
| 5xx | 服务器内部错误 |

## 限流规则

WB API 使用 **Token Bucket** 算法限流。关键响应头：

| Header | 含义 |
|--------|------|
| `X-Ratelimit-Remaining` | 当前还可立即发出的请求数 |
| `X-Ratelimit-Retry` | 被限流后多少秒后可以重试 |
| `X-Ratelimit-Limit` | Burst 最大值 |
| `X-Ratelimit-Reset` | 多少秒后 Burst 恢复到最大值 |

### 各类别典型限流（每账号）

| API 类别 | Token 类型 | 周期 | 限制 | 间隔 | Burst |
|----------|-----------|------|------|------|-------|
| Marketplace（订单/仓储） | Personal/Service | 1分钟 | 300 | 200ms | 20 |
| Marketplace | Basic/Test | 1分钟 | 150 | 200ms | 10 |
| Content（商品） | 全部 | 1分钟 | 100 | 600ms | 5 |
| 评价/问答 | Personal/Service | 1秒 | 3 | 333ms | 6 |
| 评价/问答 | Basic | 1小时 | 5 | 12分钟 | 1 |
| 财务 | Personal/Service | 1分钟 | 1 | 1分钟 | 1 |
| 财务 | Basic | 24小时 | 1 | 24小时 | 1 |

> **注意**：409 错误的请求按 5~10 个请求计数，`X-Ratelimit-Remaining` 会对应减少。

## 通用响应格式

大多数 API 返回如下结构：

```json
{
  "data": [...],
  "error": false,
  "errorText": "",
  "additionalErrors": null
}
```

带分页的接口通常用 `cursor` 或 `next` 游标分页，每次最多返回 1000 条。

## API 分类与 Token 对应

| Token 类别名称 | 覆盖 API |
|---------------|---------|
| Контент (内容) | 商品卡片、分类、媒体、标签 |
| Маркетплейс (市场) | 订单 FBS/DBS/Click-collect、仓储、库存、价格 |
| Аналитика (分析) | 销售漏斗、搜索报告、库存报告 |
| Реклама (广告) | 广告活动、促销日历 |
| Вопросы и отзывы (问答) | 评价、问题、聊天、退货 |
| Финансы (财务) | 余额、对账单、采购单 |
| Статистика (统计) | 旧版报表接口（部分已 Deprecated） |

## 通用分页参数

```
GET /api/v3/xxx?limit=1000&next=0
```

- `limit`：每页数量，最大 1000
- `next`：游标值，初始为 0，从上一次响应中获取下一个值
