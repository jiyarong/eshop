# 05 促销活动

> **模块说明**：管理 Ozon 平台促销活动——查询可参与的活动、活动中的商品、申请/退出活动等。

共 **8** 个接口。

| # | 接口名称 | HTTP 方法 |

| 2 | [获取活动中的商品列表](#2-获取活动中的商品列表) | `POST /v1/promos/products` |

| 7 | [获取活动日历](#7-获取活动日历) | `POST /v1/promos/calendar` |
| 8 | [获取活动类别](#8-获取活动类别) | `POST /v1/promos/category` |

## 1. 获取促销活动列表

```
POST /v1/promos/list
```

获取当前可参与或正在进行中的 Ozon 平台促销活动列表。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `filter` | object | ✅ | 筛选条件 |
| `limit` | integer | | 每页数量 |
| `last_id` | string | | 分页游标 |

**filter 子字段：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| `type` | string | 活动类型：`PROMO`（促销）/ `ACTIONS`（营销活动） |
| `status` | string | 活动状态：`ACTIVE` / `PLANNED` / `FINISHED` |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | array[object] | 活动列表 |
| `result[].id` | string | 活动 ID |
| `result[].title` | string | 活动名称 |
| `result[].type` | string | 活动类型 |
| `result[].starts_at` | string (datetime) | 开始时间 |
| `result[].ends_at` | string (datetime) | 结束时间 |
| `result[].status` | string | 活动状态 |
| `result[].products_count` | integer | 参与商品数量 |

### 请求示例

```http
POST /v1/promos/list
Content-Type: application/json

{
  "filter": { "type": "PROMO", "status": "ACTIVE" },
  "limit": 50
}
```

## 2. 获取活动中的商品列表

```
POST /v1/promos/products
```

获取指定促销活动中已加入的商品列表及商品的活动信息。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `promo_id` | string | ✅ | 活动 ID |
| `limit` | integer | | 每页数量 |
| `last_id` | string | | 分页游标 |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | array[object] | 活动商品列表 |
| `result[].product_id` | integer | 商品 ID |
| `result[].offer_id` | string | 商品货号 |
| `result[].price` | number | 活动价格 |
| `result[].old_price` | number | 原价 |
| `result[].discount` | number | 折扣百分比 |

---

## 3. 获取可参与活动的商品

```
POST /v1/promos/products/available
```

查询指定活动下卖家可参与的商品列表（满足活动条件但尚未加入的商品）。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `promo_id` | string | ✅ | 活动 ID |
| `limit` | integer | | 每页数量 |

---

## 4. 将商品添加到活动

```
POST /v1/promos/products/add
```

将符合条件的商品添加到指定的促销活动中。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `promo_id` | string | ✅ | 活动 ID |
| `products` | array[object] | ✅ | 商品列表 |
| `products[].product_id` | integer | ✅ | 商品 ID |
| `products[].price` | number | | 活动价格 |

### 响应

- **200**: 添加成功
- **default**: 错误（含失败商品列表）

---

## 5. 从活动中删除商品

```
POST /v1/promos/products/delete
```

将商品从指定促销活动中移除。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `promo_id` | string | ✅ | 活动 ID |
| `product_ids` | array[integer] | ✅ | 要移除的商品 ID 列表 |

---

## 6. 获取活动申请列表

```
POST /v1/promos/request/list
```

查询卖家提交的参与活动的申请记录及其状态。

---

## 7. 获取活动日历

```
POST /v1/promos/calendar
```

获取平台促销活动日历，显示各时间段的活动安排。

---

## 8. 获取活动类别

```
POST /v1/promos/category
```

获取促销活动的类别/标签列表，用于了解不同类型活动的适用范围。
