# Ozon Seller API 文档 — 模块概览

> **API 端点**：`https://api-seller.ozon.ru`
> **认证方式**：`Client-Id` + `Api-Key` 请求头

## 业务场景导航

### 商品上架流程

```
02-类目与属性 → 查询类目和属性
03-商品管理   → 创建/校验商品 → 上传图片 → 查询导入状态
04-库存与价格 → 设置库存 → 设置价格
```

### 订单处理流程（FBS / rFBS）

```
08-订单与货件 → 查询新订单 → 备货(拣货打包) → 打印标签
            → 创建发运 → 打印货位标签 → 确认发运
            → 更新配送状态(运输中→送达)
            → 管理通行证(入场证)
```

### 退货处理流程

```
09-退货管理 → 查询退货申请 → 查看退货原因
          → 批准/拒绝退货 → 跟踪退货运单
10-取消订单 → 处理条件取消申请
```

### 财务对账流程

```
13-财务 → 查交易总额 → 查交易流水 → 查结算报告 → 查费用明细 → 查余额
12-报告 → 生成财务报告下载
```

## 通用约定

| 项目 | 说明 |
|------|------|
| 请求方式 | 所有接口均为 `POST`，参数通过 JSON Body 传递 |
| 认证头 | `Client-Id: <your_client_id>` / `Api-Key: <your_api_key>` |
| 响应格式 | `application/json` |
| 分页方式 | 多为游标分页（`last_id` / `cursor`），部分为 `offset` + `limit` |
| 批量上限 | 通常在 20~1000 不等，见各接口说明 |
| 异步操作 | 商品导入（`v2/product/import`）、报告生成等需轮询状态接口 |
| 语言参数 | `DEFAULT`（默认俄语）/ `RU` / `EN` |

## 版本差异

部分接口存在多个版本（v1 ~ v5），主要差异：

- **v3 / v4 / v5**：通常为最新稳定版，推荐优先使用
- **v2**：功能增强版，增加更多筛选和响应字段
- **v1**：基础版，部分已标记过时（deprecated）

建议：
- 货件列表：优先用 `v3/posting/fbs/list`
- 货件详情：优先用 `v3/posting/fbs/get`
- 商品创建：优先用 `v2/product/import`
- 商品校验：优先用 `v2/product/validation`
- 库存更新：优先用 `v3/product/stocks`
- 库存查询：使用 `v1/product/info/stocks`
- 财务交易：优先用 `v3/finance/transaction/list`
