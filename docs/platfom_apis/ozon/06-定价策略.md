> **模块说明**：管理定价策略的全生命周期——创建策略、添加/删除商品、激活/停用策略。通过定价策略可以实现自动价格调整。

## 核心概念

定价策略（Pricing Strategy）支持以下定价模式：

| 模式 | 说明 |
|------|------|
| `MIN_PRICE` | 最低限价——设置最低售价，不低于竞争对手价格的一定比例 |
| `COMPETITOR` | 竞品跟价——自动调整价格以匹配或低于竞品 |
| `FIXED` | 固定折扣——按固定比例降低或提高价格 |

策略状态生命周期：`NOT_ACTIVE` → `ACTIVE`（激活后生效）→ `NOT_ACTIVE`（可停用）

## 1. 创建定价策略

```
POST /v1/pricing-strategy/create
```

创建一个新的定价策略。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `name` | string | ✅ | 策略名称 |
| `type` | string | ✅ | 策略类型：`MIN_PRICE` / `COMPETITOR` / `FIXED` |
| `comment` | string | | 备注说明 |

### 响应

```json
{ "result": { "pricing_strategy_id": "str_abc123" } }
```

---

## 2. 获取定价策略列表

```
POST /v1/pricing-strategy/list
```

获取所有定价策略列表，支持按名称和状态筛选。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `filter` | object | | 筛选条件 |
| `filter.name` | string | | 策略名称（模糊匹配） |
| `filter.status` | string | | 策略状态 |
| `limit` | integer | ✅ | 每页数量 |
| `last_id` | string | | 分页游标 |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `strategies` | array[object] | 策略列表 |
| `strategies[].pricing_strategy_id` | string | 策略 ID |
| `strategies[].name` | string | 策略名称 |
| `strategies[].type` | string | 策略类型 |
| `strategies[].status` | string | 状态 |
| `strategies[].products_count` | integer | 关联商品数量 |
| `strategies[].created_at` | string (datetime) | 创建时间 |

## 4. 将商品添加到策略

```
POST /v1/pricing-strategy/products/add
```

将一批商品添加到定价策略中，并为每个商品设置策略参数。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `pricing_strategy_id` | string | ✅ | 策略 ID |
| `products` | array[object] | ✅ | 商品列表 |
| `products[].product_id` | integer | ✅ | 商品 ID |
| `products[].offer_id` | string | | 商品货号 |
| `products[].price` | number | | 策略价格（依策略类型而定） |
| `products[].min_price` | number | | 最低限价 |
| `products[].max_price` | number | | 最高限价 |

### 请求示例

```http
POST /v1/pricing-strategy/products/add
Content-Type: application/json

{
  "pricing_strategy_id": "str_abc123",
  "products": [
    {
      "product_id": 123456,
      "min_price": 500,
      "max_price": 1500
    }
  ]
}
```

### 响应

```json
{
  "result": {
    "added": [123456],
    "errors": []
  }
}
```
