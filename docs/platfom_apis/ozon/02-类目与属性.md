# 02 类目与属性

> **模块说明**：查询 Ozon 平台的商品类目树、类目属性和属性可选值，是创建商品前的必要准备步骤。

共 **4** 个接口。

---

## API 列表

| # | 接口名称 | HTTP 方法 |
|---|---------|----------|
| 1 | [获取类目树](#1-获取类目树) | `POST /v1/description-category/tree` |
| 2 | [获取类目属性列表](#2-获取类目属性列表) | `POST /v1/description-category/attribute` |
| 3 | [获取属性可选值](#3-获取属性可选值) | `POST /v1/description-category/attribute/values` |
| 4 | [获取类目限制信息](#4-获取类目限制信息) | `POST /v1/description-category/limit` |

---

## 1. 获取类目树

```
POST /v1/description-category/tree
```

获取 Ozon 平台的完整商品类目树结构（父子层级关系）。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `language` | string | | 响应语言：`RU`（俄语）或 `EN`（英语），默认为 `DEFAULT` |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | array[object] | 类目树根节点列表 |
| `result[].category_id` | integer | 类目 ID |
| `result[].title` | string | 类目名称 |
| `result[].children` | array[object] | 子类目列表（递归结构） |

### 请求示例

```http
POST /v1/description-category/tree
Content-Type: application/json

{
  "language": "DEFAULT"
}
```

### 响应示例

```json
{
  "result": [
    {
      "category_id": 17028922,
      "title": "电子产品",
      "children": [
        {
          "category_id": 17038062,
          "title": "手机配件",
          "children": []
        }
      ]
    }
  ]
}
```

## 2. 获取类目属性列表

```
POST /v1/description-category/attribute
```

获取指定类目下的所有属性定义（包括必填属性和可选属性），以及属性的类型、是否多选等信息。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `description_category_id` | integer | ✅ | 类目 ID |
| `language` | string | | 语言：`RU` / `EN` / `DEFAULT` |
| `type_id` | integer | | 商品类型 ID（进一步细化属性） |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | array[object] | 属性列表 |
| `result[].id` | integer | 属性 ID（创建商品时使用） |
| `result[].name` | string | 属性名称 |
| `result[].description` | string | 属性说明 |
| `result[].type` | string | 属性类型（如 `string`、`dictionary`、`number`） |
| `result[].is_collection` | boolean | 是否允许多值 |
| `result[].is_required` | boolean | 是否必填 |
| `result[].group_id` | integer | 属性分组 ID |
| `result[].dictionary_id` | integer | 字典 ID（如果属性值来自字典） |

### 请求示例

```http
POST /v1/description-category/attribute
Content-Type: application/json

{
  "description_category_id": 17038062,
  "language": "DEFAULT"
}
```

### 响应示例

```json
{
  "result": [
    {
      "id": 85,
      "name": "品牌",
      "description": "商品的品牌名称",
      "type": "dictionary",
      "is_collection": false,
      "is_required": true,
      "group_id": 0,
      "dictionary_id": 1000
    },
    {
      "id": 8229,
      "name": "商品类型",
      "type": "dictionary",
      "is_collection": false,
      "is_required": true,
      "dictionary_id": 2000
    },
    {
      "id": 9048,
      "name": "型号名称",
      "type": "string",
      "is_collection": false,
      "is_required": false
    }
  ]
}
```

## 3. 获取属性可选值

```
POST /v1/description-category/attribute/values
```

获取字典类型属性的可选值列表。当一个属性类型为 `dictionary` 时，需要通过此接口获取可用的值。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `attribute_id` | integer | ✅ | 属性 ID |
| `description_category_id` | integer | ✅ | 类目 ID |
| `language` | string | | 语言 |
| `last_value_id` | integer | | 分页：上一页最后一个值的 ID |
| `limit` | integer | | 每页数量（默认 1000，最大 5000） |
| `type_id` | integer | | 商品类型 ID |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | array[object] | 属性值列表 |
| `result[].id` | integer | 属性值 ID（`dictionary_value_id`） |
| `result[].value` | string | 属性值文本 |
| `has_next` | boolean | 是否有下一页 |

### 请求示例

```http
POST /v1/description-category/attribute/values
Content-Type: application/json

{
  "attribute_id": 85,
  "description_category_id": 17038062,
  "limit": 100
}
```

### 响应示例

```json
{
  "result": [
    { "id": 5060050, "value": "Samsung" },
    { "id": 39638, "value": "Xiaomi" },
    { "id": 971034861, "value": "Apple" }
  ],
  "has_next": false
}
```

## 4. 获取类目限制信息

```
POST /v1/description-category/limit
```

查询类目级别的商品创建数量限制及已用额度。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `language` | string | | 语言 |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | array[object] | 各品类限制信息 |
| `result[].category_id` | integer | 类目 ID |
| `result[].limit` | integer | 该类目可创建的商品总数上限 |
| `result[].usage` | integer | 已使用的数量 |
| `result[].title` | string | 类目名称 |

### 请求示例

```http
POST /v1/description-category/limit
Content-Type: application/json

{
  "language": "DEFAULT"
}
```
