# 01 账户与认证

> **模块说明**：管理 API 密钥和卖家账户信息，是所有 API 调用的基础认证模块。

共 **3** 个接口。

---

## API 列表

| # | 接口名称 | HTTP 方法 |
|---|---------|----------|
| 1 | [获取 API 密钥列表](#1-获取-api-密钥列表) | `POST /v1/api-key/list` |
| 2 | [创建 API 密钥](#2-创建-api-密钥) | `POST /v1/api-key/create` |
| 3 | [获取卖家账户信息](#3-获取卖家账户信息) | `POST /v1/seller/info` |

---

## 1. 获取 API 密钥列表

```
POST /v1/api-key/list
```

获取当前卖家账户下所有已创建的 API 密钥（包含 `Client-Id` 和 `Api-Key`）。

### 请求参数

无请求体。

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | object | 返回结果容器 |
| `result.api_keys` | array[object] | API 密钥列表 |
| `result.api_keys[].id` | integer | 密钥 ID |
| `result.api_keys[].name` | string | 密钥名称（创建时指定） |
| `result.api_keys[].client_id` | string | Client-Id（用于 API 认证） |
| `result.api_keys[].created_at` | string (datetime) | 密钥创建时间 |
| `result.api_keys[].expires_at` | string (datetime) | 密钥过期时间，永久有效时为空 |

### 请求示例

```http
POST /v1/api-key/list
Content-Type: application/json

{}
```

### 响应示例

```json
{
  "result": {
    "api_keys": [
      {
        "id": 12345,
        "name": "MyAppKey",
        "client_id": "123456",
        "created_at": "2024-01-15T10:30:00Z",
        "expires_at": null
      }
    ]
  }
}
```

## 2. 创建 API 密钥

```
POST /v1/api-key/create
```

为当前卖家账户创建新的 API 密钥。创建成功后请立即保存 `Api-Key` 值，**Ozon 仅展示一次**。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `name` | string | ✅ | 密钥名称（用于标识用途） |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | object | 返回结果容器 |
| `result.id` | integer | 密钥 ID |
| `result.name` | string | 密钥名称 |
| `result.client_id` | string | Client-Id |
| `result.api_key` | string | ⚠️ Api-Key 值（**仅展示一次，请立即保存**） |
| `result.created_at` | string (datetime) | 创建时间 |

### 请求示例

```http
POST /v1/api-key/create
Content-Type: application/json

{
  "name": "库存同步脚本"
}
```

### 响应示例

```json
{
  "result": {
    "id": 12346,
    "name": "库存同步脚本",
    "client_id": "123456",
    "api_key": "eyJhbGciOi...（仅展示一次）",
    "created_at": "2024-06-01T08:00:00Z"
  }
}
```

## 3. 获取卖家账户信息

```
POST /v1/seller/info
```

获取当前认证卖家的基本账户信息，包括账户名称、状态、注册国家等。

### 请求参数

无请求体。

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | object | 返回结果容器 |
| `result.name` | string | 卖家账户名称（公司名） |
| `result.inn` | string | 纳税人识别号（俄罗斯 ИНН） |
| `result.kpp` | string | 税务登记原因代码 |
| `result.country_code` | string | ISO 国家代码 |
| `result.state` | string | 账户状态（如 `enabled`） |
| `result.timezone` | string | 账户所在时区 |

### 请求示例

```http
POST /v1/seller/info
Content-Type: application/json

{}
```

### 响应示例

```json
{
  "result": {
    "name": "ООО «Ромашка»",
    "inn": "7701234567",
    "kpp": "770101001",
    "country_code": "RU",
    "state": "enabled",
    "timezone": "Europe/Moscow"
  }
}
```
