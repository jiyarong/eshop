## 1. 获取退货列表（rFBS）

```
POST /v1/returns/list/rfbs
```

按货件号（posting_number）或状态查询 rFBS 退货单列表，支持分页。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `filter` | object | ✅ | 筛选条件 |
| `last_id` | integer | | 分页游标 |
| `limit` | integer | ✅ | 每页数量，最大 1000 |

**filter 子字段：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| `posting_number` | array[string] | 货件编号列表 |
| `status` | string | 退货状态：`ALL` / `ON_WAY_TO_SUPPLIER` / `ACCEPTED` / `CANCELLED` / `MOVING` |
| `accepted_from_customer_moment` | object | 按客户接受退货的时间范围筛选（含 `from`/`to` datetime） |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `has_next` | boolean | 是否有下一页 |
| `last_id` | integer | 分页游标 |
| `returns` | array[object] | 退货单列表 |
| `returns[].id` | integer | 退货单 ID |
| `returns[].company_id` | integer | 公司/仓库 ID |
| `returns[].posting_number` | string | 货件编号 |
| `returns[].product_name` | string | 商品名称 |
| `returns[].quantity` | integer | 退货数量 |
| `returns[].price` | number | 商品价格 |
| `returns[].status` | string | 退货状态 |
| `returns[].accepted_from_customer_moment` | string (datetime) | 客户退货时间 |
| `returns[].return_reason` | object | 退货原因（`id` + `name`） |
| `returns[].returned_to_ozon_moment` | string (datetime) | 商品返回 Ozon 仓库时间 |

### 请求示例

```http
POST /v1/returns/list/rfbs
Content-Type: application/json

{
  "filter": { "status": "ACCEPTED" },
  "limit": 100
}
```

## 3. 获取退货原因列表

```
POST /v1/returns/reasons
```

获取 Ozon 平台的退货原因字典（如 "商品损坏"、"买家改变主意" 等）。

### 请求参数

无请求体。

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `reasons` | array[object] | 退货原因列表 |
| `reasons[].id` | integer | 退货原因 ID |
| `reasons[].name` | string | 退货原因名称 |

### 请求示例

```http
POST /v1/returns/reasons
Content-Type: application/json

{}
```

## 4. 批准退货

```
POST /v1/returns/approve
```

同意指定退货单的退货请求，同意后 Ozon 将安排退回物流。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `return_id` | integer | ✅ | 退货单 ID |
| `comment` | string | | 备注 |

### 请求示例

```http
POST /v1/returns/approve
Content-Type: application/json

{
  "return_id": 123456789,
  "comment": "同意退货"
}
```

## 6. 获取退货详情

```
POST /v1/returns/info
```

获取指定退货单的详细信息，包括商品信息、金额、物流和状态时间线。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `return_id` | integer | ✅ | 退货单 ID |

### 响应关键字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result.id` | integer | 退货单 ID |
| `result.posting_number` | string | 货件编号 |
| `result.items` | array[object] | 退货商品列表（含 `name`、`quantity`、`price`） |
| `result.status` | string | 当前状态 |
| `result.return_reason` | object | 退货原因 |
| `result.logistic` | object | 物流信息（含跟踪号） |
