## 核心流程

```
创建商品:   属性查询 → v2/import 创建 → import/info 查状态 → 上传图片 → 上架
更新商品:   product/update
下架商品:   product/archive（归档）
```

## 1. 获取商品列表

```
POST /v1/product/list
```

获取卖家商品列表，支持按商品 ID、货号、状态、类目等多维度筛选和分页。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `filter` | object | ✅ | 筛选条件 |
| `last_id` | string | | 分页游标 |
| `limit` | integer | ✅ | 每页数量，最大 1000 |

**filter 子字段：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| `offer_id` | array[string] | 按货号（Seller SKU）筛选 |
| `product_id` | array[integer] | 按商品 ID 筛选 |
| `visibility` | string | `ALL` / `VISIBLE` / `INVISIBLE` / `ARCHIVED` / `READY_TO_SUPPLY` |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result.items` | array[object] | 商品列表 |
| `result.items[].product_id` | integer | 商品 ID |
| `result.items[].offer_id` | string | 卖家货号 |
| `result.items[].name` | string | 商品名称 |
| `result.items[].price` | string | 当前价格 |
| `result.items[].category_id` | integer | 所属类目 ID |
| `result.items[].visible` | boolean | 是否上架可见 |
| `result.items[].state` | string | 商品状态 |
| `result.items[].images` | array[string] | 图片 URL 列表 |
| `result.last_id` | string | 分页游标 |
| `result.total` | integer | 总数量 |

### 请求示例

```http
POST /v1/product/list
Content-Type: application/json

{
  "filter": { "visibility": "VISIBLE" },
  "limit": 50
}
```

## 2. 获取商品详情

```
POST /v1/product/info
```

获取单个商品的完整信息（不含库存、价格详见专属接口）。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `product_id` | integer | ✅ | 商品 ID |
| `offer_id` | string | | 货号（与 product_id 二选一） |
| `sku` | integer | | SKU ID |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result.id` | integer | 商品 ID |
| `result.offer_id` | string | 卖家货号 |
| `result.name` | string | 商品名称 |
| `result.description` | string | 商品描述 |
| `result.barcode` | string | 条码 |
| `result.category_id` | integer | 类目 ID |
| `result.state` | string | 商品状态 |
| `result.vat` | string | 增值税率 |
| `result.images` | array[string] | 图片 URL 列表 |
| `result.created_at` | string (datetime) | 创建时间 |
| `result.attributes` | array[object] | 属性列表 |
| `result.stocks` | object | 库存信息（含 `present`、`reserved`、`expected`） |
| `result.price` | string | 当前售价 |

## 6. 创建商品

```
POST /v2/product/import
```

创建新商品或更新已有商品。这是最常用的商品操作接口。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `items` | array[object] | ✅ | 商品列表（支持批量） |

**items 元素字段：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `offer_id` | string | ✅ | 卖家货号（唯一标识） |
| `name` | string | ✅ | 商品名称 |
| `description_category_id` | integer | ✅ | 类目 ID |
| `price` | string | ✅ | 售价 |
| `old_price` | string | | 原价/划线价 |
| `premium_price` | string | | Premium 会员价 |
| `vat` | string | ✅ | 增值税率：`0`、`0.1`、`0.2`、`vat_not_used` |
| `barcode` | string | | 商品条码 |
| `depth` | integer | ✅ | 包装深度（毫米） |
| `height` | integer | ✅ | 包装高度（毫米） |
| `width` | integer | ✅ | 包装宽度（毫米） |
| `weight` | integer | ✅ | 重量（克） |
| `dimension_unit` | string | | 尺寸单位：`mm` |
| `weight_unit` | string | | 重量单位：`g` |
| `images` | array[string] | ✅ | 图片 URL 列表（至少 1 张） |
| `primary_image` | string | | 主图 URL |
| `attributes` | array[object] | ✅ | 属性列表 |
| `attributes[].id` | integer | | 属性 ID |
| `attributes[].values` | array[object] | | 属性值 |
| `attributes[].values[].dictionary_value_id` | integer | | 字典值 ID |
| `attributes[].values[].value` | string | | 文本值 |

### 响应

```json
{
  "result": {
    "task_id": "123456789"
  }
}
```

> ⚠️ 创建是异步操作，需使用 `POST /v1/product/import/info` 查询结果。

### 请求示例

```http
POST /v2/product/import
Content-Type: application/json

{
  "items": [
    {
      "offer_id": "SKU001",
      "name": "智能手机壳 - 黑色",
      "description_category_id": 17038062,
      "price": "500",
      "old_price": "800",
      "vat": "0.2",
      "depth": 15,
      "height": 160,
      "width": 80,
      "weight": 80,
      "images": ["https://example.com/img1.jpg"],
      "attributes": [
        {
          "id": 85,
          "values": [{ "dictionary_value_id": 5060050 }]
        },
        {
          "id": 8229,
          "values": [{ "dictionary_value_id": 10000 }]
        }
      ]
    }
  ]
}
```

## 21. 商品错误校验（v2）

```
POST /v2/product/validation
```

在正式创建商品前，预先校验商品数据是否符合平台要求。返回详细的校验错误列表。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `items` | array[object] | ✅ | 待校验的商品列表（格式同 v2/import） |

### 响应

```json
{
  "result": {
    "validated": [
      { "offer_id": "SKU001", "errors": [] }
    ],
    "errors": [
      { "offer_id": "SKU002", "errors": [{ "code": "...", "message": "..." }] }
    ]
  }
}
```
