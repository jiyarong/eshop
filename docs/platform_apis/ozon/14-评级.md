# 14 评级

> **模块说明**：查询卖家的服务质量评级——错误指数（FBS / rFBS）及其影响评级的货件明细。

共 **2** 个接口。

---

## API 列表

| # | 接口名称 | HTTP 方法 |
|---|---------|----------|
| 1 | [获取错误指数](#1-获取错误指数) | `POST /v1/rating/index/fbs/info` |
| 2 | [影响错误指数的货件列表](#2-影响错误指数的货件列表) | `POST /v1/rating/index/fbs/posting/list` |

---

## 1. 获取错误指数

```
POST /v1/rating/index/fbs/info
```

获取 FBS / rFBS 模式的错误指数（Defect Index）——数值越低表示服务质量越好。同时返回该计费周期内的错误处理费用汇总。

### 请求参数

无请求体。

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `currency_code` | string | 费用币种代码 |
| `index` | number | 当前计费周期的错误指数 |
| `period_from` | string (date) | 计费周期开始日期（`YYYY-MM-DD`） |
| `period_to` | string (date) | 计费周期结束日期（`YYYY-MM-DD`） |
| `processing_costs_sum` | number | 周期内错误处理费用总额 |
| `defects` | array[object] | 按天分解的错误指数明细 |
| `defects[].date` | string (date) | 日期 |
| `defects[].index_by_date` | number | 当日错误指数 |
| `defects[].processing_costs_sum_by_date` | number | 当日错误处理费用 |

### 请求示例

```http
POST /v1/rating/index/fbs/info
Content-Type: application/json

{}
```

### 响应示例

```json
{
  "currency_code": "RUB",
  "index": 2.5,
  "period_from": "2024-06-01",
  "period_to": "2024-06-30",
  "processing_costs_sum": 1500.00,
  "defects": [
    {
      "date": "2024-06-15",
      "index_by_date": 0.5,
      "processing_costs_sum_by_date": 300.00
    },
    {
      "date": "2024-06-20",
      "index_by_date": 2.0,
      "processing_costs_sum_by_date": 1200.00
    }
  ]
}
```

## 2. 影响错误指数的货件列表

```
POST /v1/rating/index/fbs/posting/list
```

获取对当前错误指数产生影响的具体货件列表，包括错误类型、罚款金额等详细信息。支持分页。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `limit` | integer | ✅ | 每页数量，最大 1000 |
| `filter` | object | ✅ | 筛选条件 |
| `cursor` | string | | 分页游标 |

**filter 子字段：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `date_from` | string (datetime) | ✅ | 查询起始日期 |
| `date_to` | string (datetime) | ✅ | 查询截止日期 |
| `posting_numbers` | array[string] | | 按货件号筛选 |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `cursor` | string | 下一页分页游标 |
| `has_next` | boolean | 是否还有更多数据 |
| `errors` | array[object] | 影响评级的货件列表 |
| `errors[].posting_number` | string | 货件编号 |
| `errors[].error_at` | string (datetime) | 错误发生时间 |
| `errors[].index` | number | 该货件贡献的错误指数 |
| `errors[].charge_price` | number | 罚款金额 |
| `errors[].charge_price_currency_code` | string | 罚款币种 |
| `errors[].charge_percent` | number | 罚款百分比 |
| `errors[].posting_error_type` | string | 错误类型（如 `UNSPECIFIED`） |
| `errors[].delivery_schema` | string | 配送模式 |
| `errors[].product_price` | number | 商品价格 |
| `errors[].has_grace_status` | boolean | 是否处于宽限期 |

### 请求示例

```http
POST /v1/rating/index/fbs/posting/list
Content-Type: application/json

{
  "limit": 100,
  "filter": {
    "date_from": "2024-06-01T00:00:00Z",
    "date_to": "2024-06-30T23:59:59Z",
    "posting_numbers": ["41517061-0214-1"]
  }
}
```

### 响应示例

```json
{
  "cursor": "next_page_token",
  "has_next": false,
  "errors": [
    {
      "posting_number": "41517061-0214-1",
      "error_at": "2024-06-15T10:30:00Z",
      "index": 0.5,
      "charge_price": 300.00,
      "charge_price_currency_code": "RUB",
      "charge_percent": 10,
      "posting_error_type": "LATE_SHIPMENT",
      "delivery_schema": "fbs",
      "product_price": 3000.00,
      "has_grace_status": false
    }
  ]
}
```
