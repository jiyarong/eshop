## 核心关系

```
仓库 (Warehouse)  ←→  配送方式 (Delivery Method)
      │                      │
      └── 绑定关系 ──────────┘
      (warehouse delivery methods)

仓库必须绑定至少一个配送方式才能接收订单
```

## 1. 创建仓库

```
POST /v1/warehouse/create
```

创建一个新的 FBS / rFBS 仓库。包含仓库基本信息、地址、联系电话和发货时间设置。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `name` | string | ✅ | 仓库名称 |
| `address` | string | ✅ | 仓库地址 |
| `is_rfbs` | boolean | ✅ | 是否为 rFBS 仓库 |
| `phone` | string | | 联系电话 |
| `first_mile_type` | string | | 头程类型：`DROP_OFF`（自送）/ `PICK_UP`（上门揽收） |
| `working_hours` | object | | 工作时间表 |

**working_hours 子字段：**

| 参数名 | 类型 | 说明 |
|--------|------|------|
| `MONDAY` ~ `SUNDAY` | object | 每日工作时间 |
| `{day}[].from` | string (time) | 开始时间 `HH:mm` |
| `{day}[].to` | string (time) | 结束时间 `HH:mm` |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result.warehouse_id` | integer | 仓库 ID |

### 请求示例

```http
POST /v1/warehouse/create
Content-Type: application/json

{
  "name": "莫斯科中心仓",
  "address": "г. Москва, ул. Ленина, д. 1",
  "is_rfbs": true,
  "phone": "+7-999-123-45-67",
  "first_mile_type": "DROP_OFF",
  "working_hours": {
    "MONDAY": [{ "from": "09:00", "to": "18:00" }],
    "TUESDAY": [{ "from": "09:00", "to": "18:00" }]
  }
}
```

## 2. 获取仓库列表

```
POST /v1/warehouse/list
```

获取当前卖家账户下所有仓库的列表。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `limit` | integer | | 每页数量 |
| `offset` | integer | | 分页偏移 |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result` | array[object] | 仓库列表 |
| `result[].warehouse_id` | integer | 仓库 ID |
| `result[].name` | string | 仓库名称 |
| `result[].is_rfbs` | boolean | 是否 rFBS |
| `result[].created_at` | string (datetime) | 创建时间 |

### 请求示例

```http

{}
```
