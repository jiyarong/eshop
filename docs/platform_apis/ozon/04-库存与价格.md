# 04 库存与价格

> **模块说明**：管理商品的实际库存（各仓库）、售价更新、折扣设置等。包含库存查询、价格更新、批量接口等。

共 **10** 个接口。

---

| 2 | [更新商品库存](#2-更新商品库存) | `POST /v1/product/stocks` |

## 核心关系

```
库存(Stock)  库存更新 → 仓库 + 数量
价格(Price)  价格更新 → 售价/原价/会员价
折扣(Discount) 折扣设置 → 折扣价 + 促销类型
```

---

## 1. 获取商品库存信息

```
POST /v1/product/info/stocks
```

获取单个商品在各仓库的详细库存（与 03-商品管理 中的接口 5 相同，按功能归属此处）。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `product_id` | integer | ✅ | 商品 ID |
| `offer_id` | string | | 货号 |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result.items` | array[object] | 各仓库库存信息 |
| `result.items[].warehouse_id` | integer | 仓库 ID |
| `result.items[].warehouse_name` | string | 仓库名称 |
| `result.items[].present` | integer | 在库可售数 |
| `result.items[].reserved` | integer | 已预留数（已被订单占用） |
| `result.items[].expected` | integer | 预期到货数（补货在途） |
| `result.items[].sku` | integer | SKU ID |

### 请求示例

```http
POST /v1/product/info/stocks
Content-Type: application/json

{
  "product_id": 123456
}
```

### 响应示例

```json
{
  "result": {
    "items": [
      {
        "warehouse_id": 10001,
        "warehouse_name": "莫斯科中心仓",
        "present": 50,
        "reserved": 3,
        "expected": 20,
        "sku": 123456789
      }
    ]
  }
}
```

## 2. 更新商品库存

```
POST /v1/product/stocks
```

更新商品在指定仓库的库存数量。这是 rFBS 模式的库存同步接口。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `stocks` | array[object] | ✅ | 库存更新列表 |

**stocks 元素字段：**

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `warehouse_id` | integer | ✅ | 仓库 ID |
| `product_id` | integer | ✅ | 商品 ID |
| `stock` | integer | ✅ | 该仓库的可售库存数量（绝对值） |
| `offer_id` | string | | 货号（与 product_id 二选一） |

### 响应

```json
{
  "result": { "stocks": [] }
}
```

### 请求示例

```http
POST /v1/product/stocks
Content-Type: application/json

{
  "stocks": [
    { "warehouse_id": 10001, "product_id": 123456, "stock": 50 },
    { "warehouse_id": 10001, "product_id": 789012, "stock": 30 }
  ]
}
```

## 4. 更新价格

```
POST /v1/product/price
```

更新单个商品的价格（售价、原价、会员价）。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `product_id` | integer | ✅ | 商品 ID |
| `price` | string | ✅ | 新售价 |
| `old_price` | string | | 新原价（划线价） |
| `premium_price` | string | | 新 Premium 会员价 |
| `vat` | string | | 增值税率 |
| `auto_action_enabled` | string | | 自动降价：`UNKNOWN` / `ENABLED` / `DISABLED` |
| `min_price` | string | | 最低限价（自动降价时使用） |
| `currency_code` | string | | 币种（如 `RUB`） |

### 响应

```json
{
  "result": { "status": "ok" }
}
```

### 请求示例

```http
POST /v1/product/price
Content-Type: application/json

{
  "product_id": 123456,
  "price": "500",
  "old_price": "800",
  "premium_price": "450",
  "currency_code": "RUB"
}
```

## 6. 设置商品折扣

```
POST /v1/product/update/discount
```

为商品设置限时折扣价格，用于短期促销活动。折扣到期后自动恢复原价。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `discount` | number | ✅ | 折扣百分比（如 `20` 表示 8 折） |
| `product_ids` | array[integer] | ✅ | 适用商品 ID 列表 |
| `action_id` | integer | | 关联的促销活动 ID |

### 响应

```json
{
  "result": {
    "updated": [123456, 789012],
    "errors": []
  }
}
```

## 7. 获取商品折扣信息

```
POST /v1/product/info/discounted
```

查询单个商品当前的折扣状态——是否正在促销、折扣价、活动名称等。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `discounted_skus` | array[string] | ✅ | 商品 SKU 列表（格式：`"{product_id}"` 或 `"{offer_id}"`） |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `items` | array[object] | 折扣信息列表 |
| `items[].product_id` | integer | 商品 ID |
| `items[].discounted_price` | string | 折扣价格 |
| `items[].discount_percent` | number | 折扣百分比 |
| `items[].action_name` | string | 活动名称 |
| `items[].action_id` | integer | 活动 ID |

## 9. 批量导入价格

```
POST /v1/product/import/prices
```

通过异步任务批量导入/更新商品价格。适用于大量商品的价格同步。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `prices` | array[object] | ✅ | 价格列表（同批量更新价格格式） |

### 响应

```json
{
  "result": { "task_id": "987654321" }
}
```

> ⚠️ 异步操作，使用 `POST /v1/product/import/info` 查询结果。

## 10. 获取库存变更日志

```
POST /v1/product/stock/log
```

查询指定商品的历史库存变更记录（什么时间、由谁、变更了多少）。

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `product_id` | integer | ✅ | 商品 ID |
| `offer_id` | string | | 货号 |
| `limit` | integer | | 每页数量 |
| `last_id` | string | | 分页游标 |

### 响应字段

| 字段路径 | 类型 | 说明 |
|---------|------|------|
| `result.rows` | array[object] | 变更记录 |
| `result.rows[].warehouse_id` | integer | 仓库 ID |
| `result.rows[].stock_before` | integer | 变更前库存 |
| `result.rows[].stock_after` | integer | 变更后库存 |
| `result.rows[].delta` | integer | 变更量 |
| `result.rows[].changed_at` | string (datetime) | 变更时间 |
| `result.rows[].changed_by` | string | 变更来源 |
